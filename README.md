# Cockatoo_V1 - Personal AI Knowledge Assistant

Cockatoo_V1 is a 100% local AI personal assistant that transforms your document collection (PDFs, ebooks, notes, articles) into a digital second brain that you can query anytime.

## Features

- **Privacy First**: All data remains on your device
- **Zero Subscription**: Free forever, open source
- **Offline Native**: Works without internet connection
- **Easy Setup**: Install in 5 minutes, no technical configuration
- **Resource Efficient**: Lightweight for home laptops/PCs

## Supported Document Formats

- **PDF documents** (.pdf)
- **Microsoft Word** (.docx)
- **Text files** (.txt)
- **E-books** (.epub)
- **Markdown** (.md)
- **HTML web pages** (.html)
- **CSV data** (.csv)
- **Images with text** (.jpg, .png) via OCR

## Quick Start

### Installation

# Clone the repository
git clone https://github.com/yourusername/Cockatoo_V1.git
cd Cockatoo_V1

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
